查找与引用函数之INDEX函数

发表于 2020-01-21 分类于 数据分析 ， Excel

原文链接：函数INDEX基础介绍

INDEX 函数返回表格或区域中的值或值的引用。包含两种使用方式：数组形式和引用形式。

一、数组形式

想要返回指定单元格或单元格数组的值，请使用数组形式。

INDEX 函数一共有三个参数，

  • array：必需，单元格区域或数组常量
    • 如果数组只包含一行或一列，则相对应的参数 row_num 或 column_num 为可选参数
    • 如果数组有多行和多列，但只使用 row_num 或 column_num 参数，则 INDEX 返回数组中的整行或整列，且返回值也是数组
  • row_num：必需，选择数组中的某行，函数从该行返回数值。
    • 如果省略 row_num，则必须有 column_num。
  • column_num：可选参数，选择数组中的某列，函数从该列返回数值。
    • 如果省略 column_num，则必须有 row_num。
  • 如果 row_num 或 column_num 其中一个为0，则返回整列或整行的数组

二、引用形式

如果想要返回对指定单元格的引用，请使用引用形式

参数说明：

  • reference：必需，对一个或多个单元格区域的引用
  • area_num：这个参数对应 reference 中具体引用的区域。如果引用第一个区域，则参数值为1

钣金基础知识

发表于 2019-12-08 更新于 2019-12-30 分类于 冲压工艺

一、什么是钣金

钣金，一种加工工艺，钣金至今为止尚未有一个比较完整的定义。

根据国外某专业期刊上的一则定义，可以将其定义为：钣金是针对金属薄板（通常在6mm以下）一种综合冷加工工艺，包括剪、冲/切/复合、折、铆接、拼接、成型（如汽车车身）等。其显著的特征就是同一零件厚度基本一致。

比如：电脑机箱、空调外机、汽车白车身等等，都是通过钣金得到的。

netfilter 五表五链

发表于 2019-03-28 更新于 2019-04-01 分类于 Linux 运维基础 ， 日常运维

一、netfilter 简介

1.2 工作原理图

图中的椭圆框中，上面一行小写表示表，下面一行大写表示链。

二、基本概念

2.1 表（tables）：5个

iptables包含五张表（tables）：

  • raw：用于配置数据包，raw中的数据包不会被系统跟踪。
  • mangle：用于对特定的包做修改。
  • nat：nat就是网络地址转换。用于网络地址转换（如端口映射、地址映射等）。生活中的实例——路由器。
  • filter：默认表，用于过滤包。
  • security：用于强制访问控制网络规则。CentOS6中是没有的，CentOS7中增加。工作中应该用不到。

大多数情况下，我们只需要使用nat和filter。

2.2 链（chains）

表是由链组成的，链是一些按顺序排列的规则的列表。
默认情况下，任何链中都是没有规则的。可以根据我们自己的需要添加规则。

  • filter 表包含 INPUT、OUTPUT、FORWARD 3条内建的链，这3条链作用于数据包过滤过程中的不同时间点。
    • INPUT ：对进入本机的包进行过滤
    • OUTPUT ：对从本机送出去的包进行过滤
    • FORWARD ：不是本机的包，对目标地址做更改或者说是转发。
  • nat 表包含 PREROUTING、POSTROUTING、OUTPUT 3条链。
    • PREROUTING ：外网 IP 转内网 IP 。
    • POSTROUTING ：内网 IP 转外网 IP 。
    • OUTPUT ：修改从本机出去的包。

2.3 规则（rules）

数据包的过滤基于规则，规则由一个目标（数据包包匹配所有条件后的动作）和很多匹配（导致该规则可以应用的数据包所满足的条件）指定。一个规则的典型匹配事项是数据包进入的端口（例如：eth0 或者 eth1）、数据包的类型（ICMP, TCP, 或者 UDP ）和数据包的目的端口。

目标（怎么处理数据包）使用-j或者--jump选项指定。

2.4 遍历链（Traversing Chains）

开头的原理图描述了链在任何接口上收到的网络数据包是按照怎样的顺序穿过表的交通管制链：

  • 第一个路由策略包括决定数据包的目的地是本地主机（这种情况下，数据包穿过 INPUT 链），还是其他主机（数据包穿过 FORWARD 链）；
  • 中间的路由策略包括决定给传出的数据包使用哪个源地址、分配哪个接口；
  • 最后一个路由策略存在是因为先前的 mangle 与 nat 链可能会改变数据包的路由信息。

3.iptables命令语法

netfilter 规则存放 在/etc/sysconfig/iptables中。
