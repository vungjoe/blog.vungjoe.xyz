钣金基础知识

发表于 2019-12-08 更新于 2019-12-30 分类于 冲压工艺

一、什么是钣金

钣金，一种加工工艺，钣金至今为止尚未有一个比较完整的定义。

根据国外某专业期刊上的一则定义，可以将其定义为：钣金是针对金属薄板（通常在6mm以下）一种综合冷加工工艺，包括剪、冲/切/复合、折、铆接、拼接、成型（如汽车车身）等。其显著的特征就是同一零件厚度基本一致。

比如：电脑机箱、空调外机、汽车白车身等等，都是通过钣金得到的。

一、Django 内置字段类型（不包括关系字段类型）

1.1 AutoField

AutoField 是一个自动增加的整数类型字段。通常不需要自己编写它， Django 会自动帮你添加字段 id = models.AutoField(primary_key=true) 。

  • 这是一个自增字段，从 1 开始计数
  • 如果需要自行创建主键，务必将该字段设为 primary_key=true
  • Django 在模型中只允许有一个自增字段，且该字段必须为主键
