Python程序的控制结构

发表于 2019-05-10 分类于 Python3

一、顺序结构

顺序结构其实就是程序按编写的顺序，一行行往后执行的程序结构。这是最简单的程序结构。

二、分支结构

2.1 单分支结构

单分支结构是最简单的分支结构，符合条件则执行语句块中的程序，否则直接结束分支结构。

  • <条件>可以直接使用保留字 True（无条件执行）和 False（无条件跳过）。

2.2 二分支结构

二分支结构是根据条件结果选择不同的向前路径的运行方式。

  • 条件成立执行语句块1，不成立则执行语句块2。
  • 紧凑形式：<表达式1> if <条件> else <表达式2>
  • 注意这里是表达式而不是语句块？？语句和表达式的区别——说实话，没太懂。
语句和表达式的区别. https://www.zhihu.com/question/20750344 ↩

2.3 多分支结构

  • 注意多条件之间的包含关系（越向后条件越松）
  • 注意变量取值范围的覆盖关系（越向后取值范围越大）

2.4.1 条件判断操作符

三、循环结构

3.1 for in 循环（遍历循环）

for 循环的基本语法：

上述的 <列表> ，可以有多种类型，只要它能遍历，都可以放在其中：

  • 可以是字符串，for w in "python" ——将遍历字符串 python 中的每一个字符
  • 可以是列表，for s in ['a', 'b', 'c'] ——将遍历列表中的每一个元素
  • 可以是文件，for line in file ——将遍历文件的每一行
  • 可以是 range() 函数，for i in range(10)——将遍历 0-10 。

3.2 while 循环（条件循环）

while 循环的基本语法：

只要满足条件，就会执行循环中的语句块。直到条件判断为 Flase 或显式地以 break 结尾时才结束。

3.3 循环控制保留字 break 和 continue

循环控制保留字包括两个：break 和 continue。

  • break ：结束当前层的循环，继续执行后面的语句（与当前循环同层）
  • continue ：结束本次循环，继续下一次循环

示例：求非 None 值的数之和

  • 由于 None 不能进行求和运算所以，continue 在这里的作用就是不对 None 值进行求和，直接跳过了本次循环

3.4 循环的高级应用

循环的高级应用是普通循环和 else 结合使用。它有以下的几个特性：

  • 当循环没有被 break 语句退出时，执行 else 语句块
  • else 语句块作为“正常”完成循环的“奖励”
  • 这里 else 的用法与异常处理中 else 的用法相似
