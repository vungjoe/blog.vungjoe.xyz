Python字符串

发表于 2018-06-18 更新于 2019-03-05 分类于 Python3

一、字符串

1.1 什么是字符串

由0个或多个字符组成的有序字符序列，就是字符串。特点：

  • 由一对单引号或者是一对双引号表示，只能表示单行字符串
  • 由一对三单引号或者一对三双引号表示，表示多行字符串
    • 前面说过这样做是作为程序的注释。其实质就是字符串，只不过在程序中未对这个字符串有任何操作
  • 字符串是字符的字符的有序序列，可以对其中的字符进行索引

1.2 字符串中有双引号或者单引号如何表示

三种方式：

  • “这里有一个单引号（’）”，双包单
  • ‘这里有一个双引号（”）’，单包双
  • ‘’’这里既有单引号（’），也有双引号（”）’’’

二、索引与切片

2.1 字符串的序号

从左往右（正向递增），依次是0，1，2，…
从右往左（反向递减），依次是-1，-2，-3，…

如下图所示：

2.2 索引

索引是指返回字符串中单个字符（1个汉字就是一个字符，占用两个字节）

表示方法：<字符变量>[N]

2.3 切片

切片是指返回字符串中的一个字符子串

表示方法：<字符变量>[start:end:step]， 以步长 step 对 start 到 end 之间的字符串进行切片，且不包含 end 在内。

示例：

  • start、end、step 可以省掉最多2个。
    • [:N]，表示从第一个到第 N 个
    • [M:]，表示从第 M 个到最后一个
    • [::K]，对整个字符串，每隔 K 个进行切片
  • 常用表示方法：
    • 倒序：[::-1]

2.4 转义符

转义符用 “\” 进行表示，用来表示特定字符的本意

  • print("\"")，打印一个双引号
  • 表达一些不可打印字符
    • \b，回退，光标向前回退一个位置
    • \n，换行，光标移动到下一行行首
    • \r，回车，光标回到当前行行首
    • \t，制表符

三、字符串操作符

操作符 描述
a + b 连接两个字符串，无空格
n * x 或 x * n n 个字符串连接
x in s 如果 x 是s 的子串，则返回 True，否则返回 False

四、字符串处理函数

函数 描述
len(x) 返回字符串 x 的长度，x 可以是字符串，也可以是字符串变量
str(x) 将任意类型的变量 x 转换成对应的字符串类型
eval(x) 将字符串最外侧的引号去掉，剩下语句再执行
hex(x)或 oct(x) 将十进制的x 转换成十六进制或八进制小写的字符串形式
chr(x) x 为Unicode 编码，返回其对应的字符
ord(x) x 为字符，返回其对应的 Unicode 编码

五、字符串处理方法

5.1 什么是方法

方法是一个专有名词，在面向对象的 编程中，

  • 方法特指<a>.<b>()风格中的<b>()
  • 方法本身也是函数，但与<a>有关，需<a>.<b>()方可使用
  • 字符串及变量也是<a>，存在一些方法

5.2 8个方法

下面是一些一方法提供的字符串处理方法

方法 描述
str.lower()或者str.upper() 返回字符串的副本，其结果全部为小写/大写
str.split(sep) 返回字符串被分割之后的一个列表，分割符是参数 sep（默认的分隔符是空格），分割之后的部分组成一个列表。 如 "a,b,c".split(",") 结果为 ['a','b','c']
str.count(substr) 返回子串在字符串中出现的次数
str.replace(old,new) 将字符串中的 old 子串全部替换成 new
str.center(width[,fillchar]) 字符串根据宽度 width 使用 fillchar 进行填充，默认的填充符是空格
str.strip(chars) 去掉字符串左右两侧 chars 中给出的字符，默认去掉空格、换行符\n、制表符\t
iter.join(str) 在字符串str 除最后一个元素外的左右元素后面加一个字符iter，主要用于字符串分隔。
比如','.join('12345') 结果为 '1,2,3,4,5'
