Python数值类型

发表于 2018-07-20 更新于 2019-03-05 分类于 Python3

Python中最主要的数值类型有整数（int）、浮点数（float）、布尔（bool）和复数（complex），这里主要介绍整数和浮点数。

一、整数类型

1.1 一个常用的函数

pow(x,y)：求x的y次幂

1.2 四种常用的进制表示方法

四种进制表示方法

  • 十进制：无任何特殊，默认就是十进制
  • 二进制：以0B或者0b开头（b表示binary），有正有负，如0b0101、-0B0110
  • 八进制：以0o或者0O开头（O表示October），如0O1270、-0o146
  • 十六进制：以0x或者0X开头，如0x9b

二、浮点数类型

2.1 浮点数的

Python中的浮点数类型与数学中的实数概念一致

  • 带有小数点及小数的数字
  • 浮点数取值范围和精度都存在限制，但常规计算可忽略
  • 取值范围数量级约-10＾308~10＾308，精度等级10＾-16。
  • 浮点数之间运算存在不确定尾数现象，不是bug

2.2 浮点数运算的不确定尾数

前面已经提到，浮点数运算存在不确定尾数，不是bug。

为了解决浮点数的不确定尾数，使用 round() 函数来进行圆整（四舍五入）

三、复数类型

Python 中的复数与数学中的复数是相同的，用 z = a+bj 进行表示：

  • 其中 a 是实部，b 是虚部
  • j**2 = -1
  • z.real 获取实部，z.imag 获取虚部

四、布尔类型

  • 布尔类型仅有两个值—— 0 和 1。
  • 各种类型对应的真值
    • 整数、浮点数、复数：0 是“假”，所有非 0 数值都是真
    • 空字符串是“假”，其他是真
    • 空序列是“假”，非空序列是真
    • 空值表示无意义、或不知道，也是“假”

五、数字运算操作符

5.1 一元操作符

操作符 描述
x + y 加，x与y之和
x - y 减，x与y之差
x * y 乘，x与y之积
x / y 除，x与y之商，结果与数学中一样，10/3=3.3333333333333335
x // y 整数除，结果是商的整数部分。10//3=3
x % y 取余，模运算，10%3=1
x ** y x的y次幂，与函数pow(x,y)效果相同
+x x本身
-y y的负数

5.2 二元操作符

二元操作符有对应的增强赋值操作符，x op= y

二元操作符有，+=、-=、*=、/=、//=、%=、

  • 含义：x = x ** 2

5.3 数字类型之间的关系

  • 不同的数字类型可以进行混合运算，生成结果“最宽”的类型
  • 整数—>浮点数—>复数

六、数字运算函数

函数 含义
abs(x) 求绝对值
divmod(x,y) 商余，(x//y,x%y)，同时输出商和余数。比如 divmod(10,3)结果为(3,1)
pow(x,y[,z]) 幂余，(x**y)%z，
round(x[,d]) 对 x 进行四舍五入，d 表示小数位数，默认是 0（表示圆整）
max(x1,x2,…,xn) 最大值
min(x1,x2,…,xn) 最小值

七、数字类型转换函数

函数 含义
int(x) 将x变为整数，int(123.12)结果为123，int(“123”)结果为123（文本只能是整数）
float(x) 将x变成浮点数，增加小数部分，float(1) 结果为1.0，float(“1.23”) 结果为1.23
complex(x) 将x变成复数，增加虚数部分，complex(4) 结果为4+0j
